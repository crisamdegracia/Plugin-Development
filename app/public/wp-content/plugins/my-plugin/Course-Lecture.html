What are action and filter hooks

Hooks - execute an addtion piece of code -- a function - modify existing content

2 types of hooks

  • Action Hooks
    • Hooks only execute some additional code
    • They dont return anything to the function that called it.
    • without modifying existing information
  • Filters
    • Modify existing function
    • Can also erase existing return value
    • We generally pass a variable as a parameter
function test(){}

add_action('hook', 'test')
add_filter('filter', 'test')
